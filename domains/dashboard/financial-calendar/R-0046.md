# Summary

A month-view calendar grid that complements the Financial Timeline (R-0043). Bookkeepers can see the month ahead at a glance, click any day to view or add tasks, and visually scan what's coming. Tasks created in the calendar are linked to the timeline and Books Health dashboard.

# Business Context

The Financial Timeline shows events listed chronologically — good for "what's next?" thinking. But bookkeepers also need a spatial/calendar view for planning: "What does my week look like? When should I schedule the bank reconciliation? Is there a gap on Wednesday I could use to chase invoices?"

The calendar grid and the timeline are two views of the same data. Creating a task in either place makes it visible in both.

# Acceptance Criteria

## AC-1: Month grid layout

- Full-page route: `/calendar`
- Standard 7-column grid (Mon-Sun) showing the current month
- Today's cell is highlighted (subtle primary background)
- Previous and next month navigation arrows
- Current month/year displayed in header

## AC-2: Day cell content

Each day cell shows:
- Day number
- Coloured dots indicating scheduled items (max 3 dots, then "+N more")
- Dot colours match timeline categories (blue=banking, purple=payroll, amber=receivables, etc.)
- Overdue items on past days show red dots
- Completed items show green dots (smaller, faded)

## AC-3: Click day interaction

Clicking a day cell opens a slide-out panel (right side) showing:
- All tasks/events for that day (same items as the timeline)
- Each item is clickable → navigates to the relevant page
- "Add Task" button at the top of the panel
- Completed items shown with strikethrough and completion evidence

## AC-4: Add task from calendar

- Clicking "Add Task" pre-fills the due date from the selected day
- Task creation form: title, description (optional), priority (normal/high), link (optional)
- On creation, the task appears immediately in both the calendar cell and the timeline
- Same API: `POST /calendar/tasks`

## AC-5: Visual density

- Days with no items show as empty cells (clean)
- Days with 1-3 items show individual category dots
- Days with 4+ items show 3 dots + a "+N" count badge
- Weekend columns are slightly narrower or greyed
- Days outside the current month are dimmed

## AC-6: Integration with timeline

- Calendar grid and Financial Timeline share the same data source (`GET /calendar/timeline`)
- The calendar requests a 31-day (or full month) window
- Task completion in either view is reflected in both
- The timeline popover in the header remains available — the calendar page is a complementary full view

## AC-7: Responsive behaviour

- Desktop: 7-column grid with day panels
- Tablet: 7-column grid, narrower cells, dots only (no text)
- Mobile: list view grouped by day (falls back to timeline format)

# Out of Scope

- Drag-and-drop to reschedule tasks
- Week view (only month view for now)
- Multi-month view
- External calendar sync (Google/Outlook)
- Recurring task creation from calendar
